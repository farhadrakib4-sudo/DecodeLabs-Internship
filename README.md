# Rule-Based Chatbot

## Description

This is a simple rule-based chatbot built using Python. It responds to predefined user inputs using if-else statements and runs continuously until the user exits.

## Features

- Handles greetings like "hi", "hello", and "hey"
- Answers predefined questions
- Uses if-else decision-making logic
- Runs in a continuous loop
- Handles exit commands like "bye", "exit", and "quit"

## Technologies Used

- Python

## How to Run

1. Make sure Python is installed.
2. Open the terminal in the project folder.
3. Run:

```bash
python chatbot.py
